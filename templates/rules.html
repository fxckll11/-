S + V / V-(e)s S + do / does + not + V Do / Does + S + V?

Zero Conditional
Используется для описания фактов, научных истин, общеизвестных правил или привычек. Действие происходит всегда, если выполняется условие.
RuleExample
If + Present Simple,...Present Simple... f you freeze water, it becomes a solid.
First Conditional
Используется для реальных или очень вероятных ситуаций в будущем. Условие вполне может быть выполнено.
RuleExample
If + Present Simple, ... Future Simple (will/can/may + Verb) If I study, I will pass my exams easily
Second Conditional
Используется для гипотетических, маловероятных или нереальных ситуаций в настоящем или будущем. Это «мечты» или «предположения».
RuleExample
If + Past Simple, ... would + V. If I won a million dollars, I would buy a house
Third Conditional
Используется для нереальных ситуаций в прошлом. Это «сожаления» о том, что уже случилось или не случилось.
RuleExample
If + Past Perfect, ... would have + V3 (Past Participle). If I had studied harder, I would have passed the exam
Mixed Conditionals
Сочетают в себе элементы 2-го и 3-го типов. Используются, когда условие относится к прошлому, а результат к настоящему (или наоборот).
RuleExample
If + Past Perfect, ... would + V. If + Past Simple, would have + V3 (Participle II). If I had taken the map, I wouldn't be lost now. If I were you, I would have bought that car.
